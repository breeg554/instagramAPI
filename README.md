# General info
It is a Rest API for instagram app clone. 

# Tech info
- Api written in **Express**
- Authorization with **JWT**
- Live messaging with **Socket.io**
- Database is **MongoDB**
- Images stored in **Cloudinary**

# Technologies and libraries

- Node.js (Express)
- Multer
- JWT
- Morgan
- Socket.io
- MongoDB
- Cloudinary
